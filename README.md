# How to run?
- Open GitBash
- Type git clone "https://github.com/AlexMataac/Tap-Code-Challenge.git"
- Open filename "Tap-Code-Challenge"
- Run the application
- Try it with postman
- Check the data from H2 database
- Open this localhost:8081/h2-console
